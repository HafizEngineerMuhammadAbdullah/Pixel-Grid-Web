# ⚙️ How It Works

## Sidebar Toggle

The sidebar width changes dynamically using a CSS custom property:

```css
grid-template-columns: var(--sidebar-width, 250px) 1fr;
```

JavaScript updates the variable when the toggle button is clicked.
